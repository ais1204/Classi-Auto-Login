# Classi 自動ログイン拡張機能

Classiのログイン画面を開くと、自動で「Googleでログイン」→アカウント選択→（必要なら）パスワード入力まで行います。

## インストール

1. Chromeで `chrome://extensions` を開く
2. 右上の **デベロッパーモード** をオン
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. このフォルダ（`classi`）を選択

## 初期設定（最初に1回だけ）

インストールすると **設定画面が自動で開きます**。

- **メールアドレス**: 自動ログインに使うGoogleアカウント（空なら一番上のアカウントを選択）
- **パスワード**（任意）: 空欄なら手入力に任せます

入力して「保存する」を押せば完了です。
あとから変更する場合は、ツールバーの拡張機能アイコン（パズルのマーク → ピン留め推奨）をクリックすると設定画面が開きます。

これで `https://id.classi.jp/...` を開くたびに自動ログインが走ります。

## 配布について

個人情報（メール・パスワード）はソースコードには含まれず、各ユーザーが自分のChrome内（拡張機能のストレージ）に保存します。
そのため、このフォルダをそのまま配布しても、受け取った人が自分のアカウントを設定して使えます。

## 注意

- パスワードはその端末のChrome内に保存されます（暗号化はされません）。共有PCでの利用は注意してください。
- **普段はパスワード不要**です。Googleのログインが生きていればアカウント選択だけで完了します。まずは空欄で試してください。
- **2段階認証（2FA）** が表示されると、コード入力やスマホ承認は自動化できないため手動になります。
- ログインボタンやGoogleの画面の作りが変わると動かなくなることがあります。

## セキュリティ上の仕組み

Google側の自動操作は「直近10分以内にClassiのログイン画面から来た」または「ページにClassi表記がある」場合のみ動作します。普段の自分のGoogleログイン操作を勝手に行うことはありません。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `manifest.json` | 拡張機能の定義 |
| `popup.html` / `popup.js` | 設定画面（メール・パスワードの保存） |
| `background.js` | インストール時に設定画面を開く |
| `classi.js` | Classi画面で「Googleでログイン」を自動クリック |
| `google.js` | アカウント選択・メール/パスワード入力・同意の自動化 |
| `LICENSE` | ライセンス（MIT） |
